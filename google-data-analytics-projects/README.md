# Google Data Analytics Case Studies

This repository contains the three case studies I completed for the Google Data Analytics Professional Certificate.

- [Cyclistic Bike-Share](./case_study_1_bike_share) | Analyzed bike usage to recommend strategies for converting casual users into members. | R, Excel, Tableau 
- [Bellabeat Smart Devices](./case_study_2_bellabeat) | Explored smart device data to help a wellness company improve its marketing. | Python, SQL, Power BI
